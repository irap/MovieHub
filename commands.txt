
1) Creating application:

# Creating Movies symfony-based application command:
$ composer create-project symfony/framework-standard-edition Movies


cd Movies

2) Creating data base. The pdo_sqlite is used.
# Drop old DB
$ php app/console doctrine:database:drop --force

# and create a new data base. The DB settings are taken from config.yml and parameters.yml
# mkdir app\data
$ php app/console doctrine:database:create

3) User managements
#Create a bundle that will handle users. 
$ php app/console generate:bundle --namespace=Uek/UserBundle --dir=src --format=yml --no-interaction

# Create user entity at src/Uek/ProfileBundle/Entity/User.php (see Internet)
# Create/extend the data base
$ php app/console doctrine:generate:entities Uek
$ php app/console doctrine:schema:create 



============================

# Create MovieBundle. MovieBundle will be used for movies management related stuff
php app/console generate:bundle --namespace=MovieBundle --dir=src --format=yml --no-interaction

# Create Movie entity in MovieBundle 
$ php app/console doctrine:generate:entities MovieBundle/Entity/Movie

# Create/update all the database tables needed for every known entity in your application
$ php app/console doctrine:schema:update --force

# Run the server on local PC
php app/console server:run
